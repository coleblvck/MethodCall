Method Call is an Android automation application that works specifically by mapping phone numbers to a defined list of actions.

Features

- Persistent i.e.: application can be killed or phone restarted and would not interfere with automations.

Actions

- Launch an app
- Toggle flashlight
- Shutdown
- Restart
- Send sms
- Set ringer mode
- Set screen brightness
- Toggle airplane mode
- Open Url
- Play notification sound
- Toggle mobile data
- Launch activity with action
- Set volume
- Take screenshot
- Toggle auto rotate
- Toggle nfc
